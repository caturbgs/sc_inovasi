# Smart City Inovasi
Smart City Inovasi adalah aplikasi yang dibuat untuk manajemen Inovasipada setiap Suku Dinas di pemerintahan Kota Depok sehingga terintegrasi.

## Teknologi yang digunakan
* Laravel
* Bootstrap
* MySQL
